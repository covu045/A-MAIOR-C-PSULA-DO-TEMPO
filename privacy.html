Política de Privacidade

FastLife – Jejum & Hábitos

Última atualização: Março de 2026

Esta Política de Privacidade descreve como o aplicativo FastLife – Jejum & Hábitos ("Aplicativo") coleta, utiliza, armazena e protege as informações dos usuários.

Ao utilizar o aplicativo, o usuário concorda com as práticas descritas nesta política.

1. Informações coletadas

O aplicativo pode coletar determinados tipos de informações necessárias para fornecer suas funcionalidades.

1.1 Informações fornecidas pelo usuário

  • registros de períodos de jejum
  • histórico de acompanhamento
  • estatísticas de progresso
  • preferências de uso do aplicativo

Essas informações são inseridas voluntariamente pelo usuário e utilizadas apenas para fornecer as funcionalidades do aplicativo.

2. Dados relacionados à saúde

O aplicativo pode registrar informações relacionadas a hábitos de saúde e bem-estar, incluindo:

  • duração de períodos de jejum
  • histórico de jejum
  • estatísticas de acompanhamento

Esses dados são utilizados exclusivamente para permitir que o usuário acompanhe seus hábitos dentro do aplicativo.

O aplicativo não vende, compartilha ou comercializa dados de saúde.

3. Informações coletadas automaticamente

Algumas informações técnicas podem ser coletadas automaticamente para melhorar o funcionamento do aplicativo:

  • modelo do dispositivo
  • sistema operacional
  • dados de uso do aplicativo
  • identificadores anônimos do dispositivo

4. Serviços de terceiros

O aplicativo pode utilizar serviços fornecidos por terceiros.

Publicidade

O aplicativo utiliza o serviço de publicidade Google AdMob.

Esses serviços podem coletar informações anônimas para exibição de anúncios relevantes.

  • identificadores de publicidade do dispositivo
  • interações com anúncios
  • dados básicos de uso do aplicativo

Saiba mais sobre como o Google utiliza dados:

https://policies.google.com/privacy

5. Armazenamento de dados

As informações podem ser armazenadas:

  • localmente no dispositivo do usuário
  • em serviços técnicos necessários para funcionamento do aplicativo

6. Retenção de dados

Os dados são mantidos apenas pelo tempo necessário para fornecer as funcionalidades do aplicativo.

O usuário pode remover dados ao desinstalar o aplicativo ou limpar os dados do aplicativo no dispositivo.

7. Compartilhamento de informações

O aplicativo não vende ou compartilha dados pessoais dos usuários com terceiros.

Informações podem ser compartilhadas apenas quando necessário para:

  • funcionamento de serviços integrados
  • cumprimento de obrigações legais

8. Segurança

O aplicativo utiliza práticas razoáveis de segurança para proteger informações contra acesso não autorizado.

No entanto, nenhum método de transmissão ou armazenamento eletrônico é completamente seguro.

9. Privacidade de crianças

O aplicativo não é direcionado a crianças menores de 13 anos e não coleta intencionalmente dados de crianças.

10. Direitos do usuário

De acordo com legislações aplicáveis, incluindo a Lei Geral de Proteção de Dados (LGPD), o usuário pode:

  • solicitar informações sobre dados coletados
  • solicitar exclusão de dados
  • interromper o uso do aplicativo a qualquer momento

11. Consentimento

Ao utilizar o aplicativo, o usuário concorda com esta Política de Privacidade.

12. Alterações nesta política

Esta política pode ser atualizada periodicamente para refletir mudanças no aplicativo ou requisitos legais.

A versão mais recente estará sempre disponível nesta página.
